# 💙 Mental Health Support Chatbot

A fine-tuned AI chatbot that provides supportive, empathetic responses for stress, anxiety, and emotional wellness. Built as part of an AI/ML internship project.

**Note:** This chatbot is not a substitute for professional mental health support.

## How it works

- **Base Model:** DistilGPT2
- **Fine-tuned on:** EmpatheticDialogues dataset (real human empathetic conversations)
- **Training:** Google Colab with T4 GPU, using Hugging Face's Trainer API
- **Frontend:** Streamlit
- **Model hosted on:** [Hugging Face](https://huggingface.co/MARYAMSHAHEEN11/mental-health-chatbot)

## Tech Stack

- Python
- Hugging Face Transformers
- PyTorch
- Streamlit

## Skills Demonstrated

- Model fine-tuning with Hugging Face Transformers
- Emotional tone design for safe chatbots
- Conversation modeling
- Deployment using Streamlit Cloud
